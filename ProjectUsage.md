```
To use project: 
download and unzip project
in visual studio code, open terminal
in backend folder, run server.py
in a different terminal, do ollama serve(must have ollama installed)
in a different terminal, do ollama pull llama3.2:latest(or desired model, must update code to use desired model)
open project in unity
start game through unity player inside unity lab
bots will work as desired, logs can be seen in unity console and in visual studio code terminal
```
